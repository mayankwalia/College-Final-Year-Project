# Instructions
## 1. Python Backend

- Install dependencies
`
pip install -r requirements.txt
`

- Run 
`
python ./app.py
`

## 2. Vue Frontend
- Install npm
- Install node.js
- Run
`
npm install
`
- Node modules folder will be created
- `npm run serve`
- Go to `localhost:8000`
